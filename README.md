# FluxFormula

FluxFormula is a modular Unity utility suite designed with high-decoupling and automation in mind. 
This repository follows the **Unity Package Manager (UPM)** standard, with the core package residing in a subfolder.

---

## 📦 Installation

To add this package to your Unity project, use the Git URL through the Package Manager:

1. Open **Window** > **Package Manager** in Unity.
2. Click the **+** button and select **Add package from git URL...**.
3. Paste the following URL:

```text
https://github.com/twds0x13/FluxFormula.git?path=/com.twds0x13.fluxformula
